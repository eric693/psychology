Unit 02  —  認知與行為心理學

記憶的運作：
為什麼我們會遺忘？

記憶不是攝影機，而是一位說故事的人——每次回憶都是一次重新建構，而非播放錄像。本週探索記憶如何形成、如何儲存、為何遺忘，以及為何我們對自己的記憶如此有信心，卻又如此容易出錯。

一、記憶不是錄像，而是重建

多數人對記憶有一個直覺性的誤解：以為記憶像一台攝影機，忠實地錄下曾經發生的事，需要時按下播放鍵就能重現。但認知心理學的研究一再證明，這個比喻根本錯誤。

記憶是一個主動建構的過程（reconstructive process）。每次你回憶一件事，大腦並非讀取一份儲存的檔案，而是根據片段的記憶痕跡、當下的情緒狀態、既有的知識框架與後來獲得的資訊，重新拼湊出一個版本的故事。這個版本每次都可能稍有不同，且會被悄悄「填補」細節。這就是為什麼兩個人對同一件事的記憶可以相差懸殊，為什麼童年記憶往往帶有後來才有的認知，以及為什麼目擊者證詞如此不可靠。

記憶的重建性並非缺陷，而是大腦在有限儲存空間下的高效策略：不記錄所有細節，而是記錄最重要的模式與意義，並在需要時依靠推斷補全其餘。這個策略在大多數情況下運作良好，卻在需要精確記憶時暴露出驚人的脆弱。

「記憶並不像一個儲藏室，你把東西放進去，需要時再拿出來。它更像是一個維基百科頁面——你可以讀它，但你也可以隨時修改它。」

— Elizabeth Loftus，記憶研究權威，加州大學爾灣分校

二、三階段記憶模型

1968 年，Richard Atkinson 與 Richard Shiffrin 提出了影響深遠的三階段記憶模型（three-stage memory model），將記憶過程分為感覺記憶、短期記憶與長期記憶三個相互連結的儲存系統，至今仍是記憶研究的基本框架。

感覺記憶（sensory memory）是感覺訊號消逝前的極短暫保留，分為視覺影像記憶（iconic memory，約 0.5 秒）與聲音回響記憶（echoic memory，約 3–4 秒）。它的容量極大，但幾乎所有訊息都在不到一秒內消退，只有被注意到的訊號才能進入下一階段。

短期記憶（short-term memory）是暫時性的工作空間，容量極為有限。George Miller 在 1956 年的著名論文《神奇的數字：七加減二》中指出，短期記憶一次約能保持 7 ± 2 個「組塊（chunk）」的資訊——組塊是有意義的資訊單位，可以是單一數字，也可以是一個詞語或概念。電話號碼分組的習慣（如 0912-345-678 而非 0912345678），正是利用了組塊化策略來擴展有效記憶容量。未經複誦的短期記憶在 20–30 秒後就會自然消退。

長期記憶（long-term memory）是相對永久的儲存系統，容量近乎無限，可以保存數分鐘到終身。但進入長期記憶並不自動——它需要經歷編碼（encoding）的過程，將短期記憶中的訊息轉化並儲存。睡眠在這個鞏固過程中扮演關鍵角色（詳見第 3 週）。

三、工作記憶：大腦的便條紙

Atkinson-Shiffrin 模型的「短期記憶」概念後來被認為過於簡化。1974 年，Alan Baddeley 與 Graham Hitch 提出更精緻的工作記憶模型（working memory model），將短期記憶重新概念化為一個主動的多成分系統，而非單純的被動儲存槽。

工作記憶由四個成分構成：中央執行系統（central executive）是核心控制者，協調資源分配、監控任務進行、抑制無關訊息；語音迴路（phonological loop）暫時保留語音與語言訊息，是你在心中「默念」電話號碼的機制；視覺空間畫板（visuospatial sketchpad）處理視覺與空間訊息，讓你能在心中旋轉圖形或回想地圖路線；情節緩衝器（episodic buffer）則整合來自不同子系統的訊息，並與長期記憶相連。

工作記憶的容量非常有限，是認知能力的重要瓶頸。工作記憶容量高的人，在需要同時保持多個訊息並進行操作的任務（如複雜的數學推理、理解長句）上表現顯著更好。這也是為什麼「邊做事邊分心」如此損害表現——分心佔用了工作記憶資源，原本的任務就少了處理資源。

四、長期記憶的類型

長期記憶並非一個單一的儲存倉庫，而是由多個功能不同、神經基礎各異的系統組成。理解這些分類，能幫助我們更清楚地認識記憶的多樣性，也能解釋為什麼失憶症患者往往失去某些記憶，卻保留另一些。

語意記憶
Semantic Memory
對世界的一般性事實與知識，與個人經歷無關。「法國首都是巴黎」、「水的化學式是 H₂O」——這些知識你知道，但你已記不得是什麼時候學到的。
情節記憶
Episodic Memory
個人親身經歷的事件記憶，帶有時間、地點與情境脈絡。「我七歲生日那天，媽媽做了草莓蛋糕」——這是有「我」在場的記憶，Endel Tulving 稱之為心理時間旅行的能力。
程序記憶
Procedural Memory
技能與習慣的記憶，通常是非意識的、自動化的。騎腳踏車、打字、演奏樂器——這些技能一旦習得，便儲存於小腦與基底核，難以用語言描述，卻難以遺忘。
內隱記憶
Implicit Memory
無需有意識提取的記憶，包括制約反應與啟動效應（priming）。曾見過某個字詞，之後處理它的速度會更快，即使你已完全不記得見過它。這種「記憶」存在於行為中，而非意識中。
外顯記憶
Explicit Memory
需要有意識提取的記憶，涵蓋語意記憶與情節記憶。「努力回想」這個行為本身，就是在使用外顯記憶系統。海馬迴是外顯記憶形成的關鍵結構。
閃光燈記憶
Flashbulb Memory
對高度情緒化、震驚性事件的鮮明生動記憶，如重大災難、失去親人的瞬間。感覺如同照片般清晰，但研究顯示它同樣會隨時間扭曲——只是當事人對其準確性的信心特別高。

五、記憶的編碼：如何記得更牢？

並非所有的學習方式都同樣有效。Fergus Craik 與 Robert Lockhart 在 1972 年提出層次處理理論（levels of processing theory）：訊息的編碼深度決定了記憶的持久性。淺層處理（如只注意字的外形或聲音）產生脆弱的記憶痕跡；深層處理（如理解意義、聯繫個人經驗）則產生更強固的記憶。

研究支持的高效記憶策略有以下幾種，它們全都涉及更深層的編碼或更有效的提取練習：

01
間隔重複（Spaced Repetition）
將學習分散於多個時間點，而非集中於一次「填鴨式」複習。遺忘曲線研究（Ebbinghaus, 1885）顯示，在即將遺忘之前複習的效果最好。Anki 等間隔重複軟體正是基於此原理設計。
02
主動提取練習（Retrieval Practice / Testing Effect）
主動嘗試回想所學，比反覆閱讀更有效。「考試」不只是評量工具，本身就是強力的記憶鞏固行為。閉上書本，嘗試默寫剛才學到的內容，其效果遠優於再看一遍。
03
精緻化編碼（Elaborative Encoding）
將新資訊與已有知識連結，或用自己的話解釋，或創造具體的例子。「費曼技巧」——嘗試把一個概念解釋給外行人聽——正是深度精緻化編碼的實踐。
04
脈絡與狀態依賴記憶（Context-dependent Memory）
在與學習時相似的環境中提取記憶，效果更好。在考場的緊張狀態下，比在家放鬆時更容易提取在課堂中學到的內容，這也是「模擬考試環境」練習的心理學根據。
05
記憶術（Mnemonics）
利用視覺聯想、故事、韻律或空間記憶宮殿（method of loci）來組織並強化難以直接記憶的材料。記憶冠軍賽選手大多使用記憶宮殿法，將抽象資訊轉化為生動的空間旅程。

六、遺忘的原因

遺忘是記憶系統的常態，而非異常。Hermann Ebbinghaus 在 1885 年對自己進行了一系列嚴謹的記憶實驗，繪製出著名的遺忘曲線（forgetting curve）：學習後若不複習，記憶量在最初 20 分鐘內急遽下降，一小時後遺忘約 50%，一天後遺忘約 70%，之後則趨於平緩。遺忘並非線性，而是指數級的早期衰退。

遺忘原因 機制說明 日常範例
消退（Decay） 記憶痕跡因時間流逝而自然減弱，尤其在未被提取時 幾年未聯繫的熟人名字、久未使用的外語詞彙
干擾（Interference） 相似的舊記憶干擾新記憶（順向干擾），或新記憶干擾舊記憶（逆向干擾） 記混了兩個電話號碼；學了新語言後舊語言變生疏
提取失敗（Retrieval Failure） 記憶存在但缺乏足夠的提取線索，形成「舌尖現象」 名字就在嘴邊卻說不出來；聞到氣味忽然想起往事
編碼失敗（Encoding Failure） 資訊根本未被充分編碼，從未真正進入長期記憶 看了很多遍卻記不住；分心時聽過的話完全想不起
動機性遺忘（Motivated Forgetting） 對痛苦、羞恥或威脅性記憶的主動或非主動壓抑 創傷記憶的壓抑；記不住自己失禮行為的細節

七、錯誤記憶與虛假記憶

記憶不只會消逝，還會被扭曲，甚至被憑空植入。Elizabeth Loftus 是這個領域最重要的研究者，她的工作徹底改變了司法系統對目擊者證詞的認識。

Classic Research

Elizabeth Loftus 的虛假記憶研究（1974–1990s）

Loftus 在 1974 年進行了一個經典實驗：受試者觀看一段車禍影片後，被問到「兩輛車碰撞（hit）時速度有多快？」或「兩輛車猛烈撞擊（smashed into）時速度有多快？」。結果顯示，使用「smashed」的組別，估計的車速顯著更高，且一週後更可能「記得」影片中有破碎的玻璃——但影片中根本沒有玻璃破碎。

這個實驗開創了「錯誤訊息效應（misinformation effect）」的研究：事後獲得的錯誤訊息，會悄悄融入原始記憶，改變人們對事件的回憶。後來，Loftus 更進一步展示了「虛假記憶植入（false memory implantation）」：透過暗示，她成功讓約 25% 的受試者對從未發生過的兒時事件（如在購物中心迷路）產生完整、生動的「記憶」，甚至能描述細節。

Loftus 的研究對法律實踐有深遠影響。在美國，有超過 300 名死刑犯透過 DNA 檢測被證明無辜獲釋，其中約 75% 的案件涉及錯誤的目擊者證詞。記憶的可塑性不只是學術議題，它攸關真實的人命與正義。

虛假記憶的研究也對「壓抑記憶（repressed memory）」的臨床實踐提出了嚴肅的質疑。1990 年代，美國出現大量在心理治療中「恢復」兒時受虐記憶的案例，導致許多家庭破裂，部分案件進入司法程序。後來的研究顯示，許多這類「恢復的記憶」可能是在治療師的引導與暗示下被無意間植入的虛假記憶。這場「記憶戰爭」至今仍是心理學界最具爭議的議題之一，提醒我們：對自己記憶的確信感，並不等同於記憶的準確性。

Reflection Questions
Q1 回想一段你非常確信準確的童年記憶。現在思考：哪些部分可能是你後來「填補」進去的？有沒有可能你當時根本不在場，或細節與你記得的不同？這讓你有什麼感受？
Q2 「主動提取練習」的效果遠優於反覆閱讀。回顧你過去的學習習慣，你是以哪種方式學習為主？你打算如何調整你的讀書或學習方法？
Q3 Loftus 的研究顯示目擊者記憶高度不可靠，但許多司法系統仍大量依賴目擊者證詞。你認為應該如何在司法實踐中應對記憶的不可靠性？有哪些具體改革可能有幫助？
Q4 「舌尖現象」是提取失敗的典型例子——你知道你知道，但就是想不起來。你通常用什麼方式幫助自己提取這類卡住的記憶？這些方法背後有什麼心理學邏輯？
